# PCD 3D Viewer

Three.jsを使用してPCDファイルを読み込んで3D表示するビューアです。

## 機能

- PCDファイルのドラッグ&ドロップ読み込み
- ファイル選択による読み込み
- デフォルトPCDファイル読み込み
- 3Dナビゲーション（回転、ズーム、パン）
- ポイント数表示
- 軸・グリッド表示

## 使用方法

### 開発サーバーの起動

```bash
npm install
npm start
```

ブラウザで `http://localhost:8080` を開いてください。

### 操作方法

- **マウス左ドラッグ**: 回転
- **マウス右ドラッグ**: パン
- **マウスホイール**: ズーム
- **デフォルトPCDを読み込み**: ワークスペースにあるPCDファイルを読み込み
- **カメラリセット**: カメラ位置を初期状態に戻す

### ファイル読み込み

1. 「ファイルを選択」ボタンでPCDファイルを選択
2. または「デフォルトPCDを読み込み」でワークスペース内のPCDファイルを読み込み

## 対応フォーマット

- PCD (Point Cloud Data) ファイル
- ASCII形式
- RGB色情報対応

## 技術仕様

- Three.js r128
- OrbitControls
- BufferGeometry
- PointsMaterial

## 注意事項

- 大きなPCDファイルはメモリ使用量が多くなる場合があります
- ブラウザのWebGLサポートが必要です
- ローカルサーバーでの実行が推奨されます（CORS制限のため）
